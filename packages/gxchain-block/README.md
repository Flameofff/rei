# @gxchain2/block

 Definition of block structure, based on '@ethereumjs/block'`. Added logic about `WrappedBlock`.

## INSTALL

```sh
npm install @gxchain2/block
```

## USAGE

```ts
new WrappedBlock(block,true);
```
